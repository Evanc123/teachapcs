Re-assigning variables

If you're using the variable's value to set the new value, you can shorten the statement

int age = 17;
age = age + 5;              // Age = 22

to

age += 5;                    // Now, age = 22

+= is shorthand for "increment the old value of age by 5 and store the new value in age."
