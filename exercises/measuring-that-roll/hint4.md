You can also use a <word data-key="for">for loop</word> to trace through an <word data-key="array">array</word>: 

    int temps[] = {77, 72, 73, 69, 71};

    for(int i = 0; i < temps.length; i++){
        System.out.println(temps[i]);
    }

which'll print out:

    77
    72
    73
    69
    71
