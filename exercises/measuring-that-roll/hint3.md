You can put data in an <word data-key="array">array</word> using indices:
    
    int temps[] = new int[6];
    temps[0] = 77;
    temps[1] = 72;
    temps[2] = 74;
    temps[3] = 68;
    temps[4] = 71;
    ...

And use indices to access data from an array too:

    temp[0];            // returns 77
    temp[1];            // returns 72
    ...
